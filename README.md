# Vite&Gourmand_projet

## Mise en place de l'environnement de travail 

### Outils de développement

**Systeme d'exploitation** : Linux (Ubuntu 22.04)

**Editeur de code** : Visual Studio Code (extensions pour HTML, CSS, JavaScript, PHP et Symfony).

**Technologie utilisées**
    **Front-end** : HTML5, CSS (Bootstrap), JavaScript
    **Back-end** : PHP, framework Symfony, architecture MVC

**Base de données**
    **Base de données relationnelles** : MySQL
    **Base de données NoSQL** : MongoDB

**Outils**
    **Git/GitHub** : Gestion de versions
    **XAMPP** : Serveur local (Apache, PHP, MySQL)
    **phpMyAdmin** : Interface de gestion MySQL
    **Google Chrome** : Test rendu et outils dév integrés (inspection)

**Déploiement** : Heroku
   

J'ai utilisé la majorité de ces outils car ils etaient gratuits, accessibles, bien documentés et adaptés à ce projet en PHP/MySQL.
